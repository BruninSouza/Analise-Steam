Bibliotecas utilizadas

jupyter - 1.0.0
seaborn - 0.13.2
pandas - 2.1.3
matplotlib - 3.8.2
matplotlib-inline - 0.1.6
scipy - 1.13.0
scikit-learn - 1.3.0
numpy - 1.24.3
ellipsis - 3.1.45

Caso esteja dando erro ao rodar o código, instale as Bibliotecas via Promp de Comando (CMD)

pip install seaborn
pip install scikit-learn
pip install ellipsis

Caso não consiga instalar as Bibliotecas, atualize sua versão do pip no Promp de Comando (CMD)

python.exe -m pip install --upgrade pip
